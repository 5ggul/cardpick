운영 정보

본 페이지는 카드픽 사이트의 운영 주체·연락처·인프라를 투명하게 공개합니다. 회원의 신뢰는 운영자가 누구인지·어떻게 운영되는지 보이는 데서 시작한다고 봅니다.

1. 운영 주체

본 사이트는 개인 운영자가 운영하는 비영리 정보 사이트로 시작합니다. [운영자] 성명: 김동호 역할: 사이트 기획·개발·운영·콘텐츠 사업자 등록: 미등록(개인 운영) 사이트 규모가 일정 수준 이상으로 성장하거나 유료 서비스·광고·제휴를 시작하는 시점에 사업자 등록을 진행하고 본 페이지에 등록 정보를 갱신합니다.

2. 연락처

[일반·서비스 문의] admin@cardpick.kr [제휴·언론] contact@cardpick.kr [보도자료·인터뷰] press@cardpick.kr 전화·우편 응대는 운영 인력 한계로 받지 않습니다. 이메일은 영업일 기준 3일 이내 답변을 목표로 합니다.

3. 사이트 인프라

본 사이트는 다음 외부 서비스를 사용해 운영됩니다. - 정적 호스팅·CDN·DNS: Cloudflare Pages, Cloudflare DNS (미국) - 데이터베이스·인증·스토리지: Supabase (미국 본사, AWS ap-northeast-2 서울 리전) - 인증 위임: Google OAuth (미국) - 도메인 등록: 가비아 (대한민국) 각 서비스의 처리 항목·국외 이전 사항은 개인정보처리방침에 명시되어 있습니다.

4. 운영 단계

본 사이트는 단계적으로 기능을 공개합니다. [Phase 1 — 기초 인프라] 현재 단계 로그인·프로필·게시판·관심 카드·가격 알림 구독 기능 운영 중. 시세·박스 수치는 모두 화면 구성용 예시. [Phase 2 — 데이터 파이프라인] 예정 외부 마켓 시세 자동 수집과 카드 카탈로그 단계적 확장. 카드별 페이지 발행은 데이터 검증과 콘텐츠 품질 게이트 통과 시점에 순차 인덱싱. [Phase 3 — 커뮤니티·통계 강화] 예정 게시판 활성화에 따른 운영 정책 보완과 통계·도구 확장. 각 단계의 진입 시점은 운영 상황에 따라 조정될 수 있으며, 본 페이지에 진행 상황을 반영합니다.

5. 수익 모델

현재 본 사이트는 수익 모델을 운영하지 않습니다. 사이트 운영 비용은 운영자가 부담합니다. 향후 광고·제휴·후원 등 수익 모델을 도입하는 경우 본 페이지와 약관에 사전 공지합니다.

6. 권리 침해 신고

본 사이트에 게시된 콘텐츠가 회원 또는 제3자의 권리(저작권·상표권·초상권·명예·개인정보 등)를 침해한다고 판단되는 경우 admin@cardpick.kr 로 다음 정보를 포함해 신고해주세요. - 신고자 성명·연락처·권리 보유 증빙 - 침해 게시물 URL - 침해 사유 설명 운영자는 신고를 받은 날부터 7영업일 이내 검토·조치하고 결과를 통지합니다.
