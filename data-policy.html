데이터·라이선스 정책

본 페이지는 카드픽이 다루는 데이터의 출처·갱신 주기·라이선스·이용 정책을 정리합니다. 사이트 콘텐츠와 외부 마켓 데이터의 권리 관계를 명확히 하고, 회원·제3자가 본 사이트의 정보를 안전하게 활용할 수 있도록 합니다.

1. 운영 단계와 데이터 출처

본 사이트는 단계적으로 데이터 소스를 확장합니다. 현재 시안 단계의 가격·박스 수치는 모두 화면 구성용 예시이며, 운영 단계 진입 시 다음 출처의 공개 데이터를 정리·표시할 계획입니다. [예정 출처] - Pokémon TCG API (pokemontcg.io) — 카드 메타·TCGplayer 참고가·Cardmarket 참고가 - eBay Browse API (글로벌 거래완료 가격, 향후 통합) - 운영자 자체 거래 기록과 박스 개봉 보고 각 출처의 라이선스·이용 조건은 해당 사이트의 정책을 따릅니다.

2. 갱신 주기

운영 단계 진입 후 갱신 주기는 다음과 같이 운영할 예정입니다. - 카드 메타 정보: 신규 세트 발매 시 일괄 갱신 - 가격 정보: 카드 인기도에 따라 시간 단위~일 단위 - 박스 회수율: 새 개봉 보고가 누적되면 재계산 - 메타 정보: 주요 대회 이후 갱신 갱신 주기는 운영 상황에 따라 조정되며, 변경 시 본 페이지에 반영합니다.

4. 외부 데이터의 라이선스

본 사이트가 표시하는 외부 마켓 가격, 카드 이미지, 카드 이름·세트 이름·일러스트·로고는 각 권리자의 라이선스를 따릅니다. 본 사이트가 정리·표시하는 행위는 정보 제공 목적의 공정 이용을 전제로 하며, 권리자의 적법한 요청이 있을 경우 즉시 표시를 중단합니다.

5. API·데이터 다운로드

본 사이트는 현재 공개 API와 대량 데이터 다운로드를 제공하지 않습니다. 학술·언론·연구 목적의 데이터 인용 요청은 admin@cardpick.kr 로 보내주세요. 본 사이트의 데이터를 자동 수집(스크레이핑)하거나 무단 복제·재배포하는 행위는 금지합니다.

6. AI 검색 엔진·크롤러 정책

본 사이트는 적법한 AI 검색 엔진·크롤러의 접근을 허용하며, 자체 작성 콘텐츠를 인용·요약하는 경우 출처 표기를 권장합니다. 본 사이트는 /llms.txt 와 /robots.txt 에 크롤러 정책을 명시합니다. 인용 시 "카드픽 (cardpick.kr)" 로 출처를 표기해주시기 바랍니다.

7. 시안·운영 단계 구분

초기 시안 단계의 모든 가격·박스·게시판 수치는 화면 구성용 예시이며, 사용자에게 보이는 자리에 "예시" 라벨로 명시합니다. 운영 단계 진입 시 시안 표기는 해제되며, 실측 데이터로 교체됩니다.

8. 데이터 정정 요청

본 사이트가 게시한 카드 메타·가격·박스 정보에 오류가 있는 경우 admin@cardpick.kr 로 신고해주세요. 운영자는 신고 받은 날부터 5영업일 이내에 검토하고 필요 시 정정합니다.
