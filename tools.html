// 사용 방법

+--- 사용 방법 ---+

1. 위 도구 카드 중 준비됨 표시 도구를 클릭합니다.

2. 입력 필드에 값을 넣으면 결과가 즉시 갱신됩니다. 별도 "계산하기" 버튼은 없습니다.

3. 계산 입력값은 서버로 전송하지 않고 브라우저 안에서만 처리됩니다.

4. 계산 결과는 카드 상태·환율·수수료 변동으로 실제와 다를 수 있습니다. 수익을 보장하지 않습니다.
